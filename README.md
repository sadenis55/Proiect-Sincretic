# Proiect Sincretic



## Am ales problema cu turnurile Hanoi

Turnurile din Hanoi: Se dau trei tije A, B, C și N discuri de diametre diferite, stivuite inițial pe tija A în ordinea descrescătoare a diametrelor, formând un con. Să se determine secvența de mutări ale discurilor de pe tija inițială A către tija finală C, folosind B ca tijă intermediară, având în vedere următoarele restricții:
1. La fiecare mișcare se va muta un singur disc.
2. Un disc se poate plasa doar asupra unui disc mai mare ca diametru.
